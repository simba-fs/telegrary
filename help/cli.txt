telegrary = telegram + diary. It is a diary manager with build in telegram bot.

Usage: 
  telegrary [[[year] month] day]
  telegrary <command>
  telegrary <flag>

Available Commands:
  hash           Print the hash of args[2], this is used in telegram bot
  bot            Start telegram bot
  config         Edit config file
  help           Show this help
  push           Commit and push to remote(if exist)

Flags:
  -h, --help     Show this help
